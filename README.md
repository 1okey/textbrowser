## Textbrowser

### Setting up
```bash
    python3 -m venv venv

    # linux/macoss
    source ./venv/bin/activate
    # windows
    venv\Scripts\activate.bat

    pip3 install -r ./requirements.txt
```
### Running
```bash
	python3 -m textbrowser
```
### Testing
```bash
	pytest .
```
