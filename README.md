### Prerequisites
- setuptools
- wheel
- python3
- python3-venv

### Setup

```bash
	python3 -m venv venv
	source ./venv/bin/activate
	pip install -r ./requirements.txt
```

### Test
```bash
	pytest .
```

### Run
```bash
	python3 ./src/main.py
```
